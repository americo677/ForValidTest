
#   Desarrollador iOS

Se propone utilizar el API https://pokeapi.co  para crear una Pokédex, para el diseño visual puede utilizar de guía el archivo adjunto (Sketch) pero el candidato tiene libre elección del diseño, el nivel de detalle de cada vista, amplitud y alcance de la app debe ser el que usted crea conveniente para evaluar su nivel como desarrollador iOS, recordando que el tiempo es de solo 48 horas.

##  Datos importantes que debe tener en cuenta.

### Lenguaje
Swift / Objective C

### IDE
XCode

### Documentación API
https://pokeapi.co/docs/

### Diseño
Pokedex app - iPhone X by Daniel Motta (Link)

### Alcance
Lo ideal es que tenga las siguientes vistas, pero no tiene ni un mínimo ni un máximo:

* Splash
* Home ( Lista / Grid )
* Vista detalle del Pokémon.
* Búsquedas y/o filtros de Pokemones.

### Otros
* El archivo de diseño se puede abrir en programas como Sketch o AdobeXD 
* Él envió de la prueba debe ser compartiendo la URL del Repositorio 

### Importante
* En caso de no terminar la prueba y creas que no terminaste, no te desanimes y envíanos lo que completaste (URL), aquí te evaluaremos sobre lo realizado.
